Overview

The Express Template Site project is organized into several directories to facilitate a clear and maintainable codebase.

Directory Structure

  • controllers: Contains modules for handling different routes and business logic.
  • views: Contains HTML templates (using Nunjucks) for rendering pages.
  • public: Contains static assets such as CSS stylesheets, images, and client-side JavaScript files.
  • pages: Contains route definitions and controllers for different sections of the site.
  • index.js:: Entry point of the application where Express server is configured and routes are defined.
  • .env:: Configuration file for environment variables.

Key Files

  • index.js: Main entry point of the application where the Express server is configured.
  • HomeController.js: Controller for handling requests related to the home page.
  • Error.html: Template for displaying error messages.
  • global.css: Global stylesheet used across the site.

Code Organization

The code follows a modular approach with separate modules for routing, controllers, and views to maintain separation of concerns.

Adding a New Route: Tutorial

In this tutorial, we'll demonstrate how to add a new route to the Express Template Site.

  1. Open `pages/Home.js` and add a new route for the "About Us" page:
    2. // pages/Home.js

import { Router } from "express";
import { HomeController } from "../controllers/HomeController.js";

export const HomeRouter = Router();

HomeRouter.get('/', HomeController.Home);

// Add new route for About Us page
HomeRouter.get('/about-us', HomeController.AboutUs);
  2. Open `controllers/HomeController.js` and define the controller method for the "About Us" page:
    3. // controllers/HomeController.js

export class HomeController {
    static async Home(req, res, next) {
        try {
            res.render("home/index");
        } catch (error) {
            next(error);
        }
    }

    // Controller method for About Us page
    static async AboutUs(req, res, next) {
        try {
            res.render("home/about-us");
        } catch (error) {
            next(error);
        }
    }
}
  3. Create a new HTML template file `about-us.html` in the `views/home` directory:
    4. <!-- views/home/about-us.html -->

<h1>About Us</h1>
<p>Welcome to the About Us page!</p>
  4. Usage Example: Now you can navigate to /about-us on your Express Template Site to view the "About Us" page.

Congratulations! You have successfully added a new route for the "About Us" page to your Express Template Site.


For more information, you can refer to the documentation of Express.js and Nunjucks to explore advanced features and best practices.
